# Twitter Schrader

This is a basic twitter clone app to view, compose, favorite, and retweet tweets. It interacts with the twitter API to make a fully functional app.

![](walkthrough.gif)

## Features
 - Like
 - Unlike
 - Tweet
 - Retweet
 - Infinite Scroll
 - Login
 - Logout
